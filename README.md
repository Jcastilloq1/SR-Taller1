# SR-Taller1

En esta entrega se presenta el sistema de recomendación elaborado por Pablo Hoyos, Alejandro Lancheros, Juan Castillo y Francisco Santamaría.
Se presenta una carpeta comprimida, con un .py de preprocesamiento. Dentro de la carpeta "movie_recommender" se encuentran los modelos, la ejecución de la pagina web y el entorno de procesamiento.

Ademas de ello, en la carpeta "workshop-interface" se encuentra el front de la solución.
